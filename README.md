# TheoEd Mission — English Homepage

A complete English homepage for **TheoEd Mission (TEM)**, built from the TheoED
introduction deck and styled entirely with the **TEM design system** (see the
project root: `README.md`, `colors_and_type.css`, `assets/`).

This folder is **self-contained** — it carries its own copy of the tokens and the
images it uses, so you can publish or move it independently of the design system.

## Files
| File | What it is |
|---|---|
| `TheoEd Mission Homepage.html` | The homepage (edit this). References `homepage.css`, `colors_and_type.css`, and `assets/`. |
| `TheoEd Mission Homepage (standalone).html` | **Single self-contained build** — everything inlined. Upload this one file (or rename it `index.html`) to publish anywhere. |
| `homepage.css` | Page layout & section styling. |
| `colors_and_type.css` | Local copy of the design-system tokens. |
| `assets/` | Logo emblems + placeholder hero image and sample QR. |

## Sections
Hero (Genesis 49:22) · Mission (Source / Growth / Impact) · Why TEM · Four

Contextualized Curriculum, Excellence Standard) · Get Involved (Pray / Give /
Serve) · Faculty Learning Community · Contact footer.
